### En:
#### Task 3
Create a web application where the user can customize the design of their bank card ‚Äî specifically: enter text, choose a color from a dropdown menu, and see the final card design.

üìÑ Steps:
- Create a new file with the .html extension.
- Write HTML code to create the following elements:
- An input field for entering text
- A select element with three color options: red, blue, green
- A p element with the ID card that will display the bank card

- Write JavaScript code to implement the following behavior:
- Add an input event listener to the text field. When the text changes, update the content of the card element.
- Add a focus event listener to the text field. When the field gains focus, change its style (e.g., border and background color).
- Add a blur event listener to the text field. When the field loses focus, revert its style to default.
- Add a change event listener to the select element to update the background color of the card element based on the selected value.
- Apply CSS styling to the elements for better visual presentation.
- Save the file and open it in a browser. You can now enter text, choose a color, and see your customized bank card.
